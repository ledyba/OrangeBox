Meso - Simple Battery Indicator
=======
Android端末のバッテリーをNotificationとして通知してくれるアプリケーションです。

普通の端末では１%単位では残量がわかりませんが、このアプリを使うと分かります。

In app purchaseも実装し、寄付を行うことができます。

実用に耐えると思いますが、開発目的自体は課金APIのテストです。
